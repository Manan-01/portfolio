Event Management System

The Event Management System is a console-based desktop application written in C++. It streamlines the registration and administration of events and was developed as a part of my academic coursework. The solution uses object-oriented programming principles and file handling for persistent storage.

🧰 Core Features

  • Event Creation: Add events with name, date, venue, and max capacity.
  • Participant Registration: Register individuals to events and store securely in structured files.
  • Data Storage: Uses fstream to read/write event and participant data.
  • Search & Listing: Admin can view all participants per event or search events by ID or name.
  • Modular OOP Design: Classes for Events and Participants simplify expansion.

🛠️ Technologies Used

  • Language: C++
  • Library: Standard C++ Library (iostream, fstream, etc.)
  • Design: OOP concepts such as inheritance, encapsulation, abstraction

🌟 Objective

Designed to mimic a real-world command-line event registration tool, this project enhances core programming logic, user interface handling via CLI, and structured file storage.

📌 Learning Outcomes

  • Data persistence using text files
  • Structured, object-oriented programming
  • Input validation and menu-driven navigation
  • Implementation of reusable functions and logic blocks
  • Debugging file I/O and memory logic

🚀 Future Enhancements

  • Add GUI with frameworks like Qt or C++/CLI
  • Support exporting data to Excel or CSV
  • Enable update and delete functionality for events
  • Filter and sort events by date or participant count
